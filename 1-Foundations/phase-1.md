# Phase 1: Foundations - Python Notes 🚀
---

## 1. Python Basics 🐍

- **Data Types**:  
  `int`, `float`, `str`, `bool`, `list`, `tuple`, `set`, `dict`

- **Operators**:  
  - Arithmetic: `+`, `-`, `*`, `/`, `%`, `//`, `**`  
  - Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`  
  - Logical: `and`, `or`, `not`  

- **Control Flow**:  
  - Conditional statements: `if`, `elif`, `else`  
  - Loops: `for`, `while`  
  - Loop controls: `break`, `continue`, `pass`

---

## 2. Functions and Modules ⚙️

- **Functions**: reusable and modular code.

```python
def function_name(parameters):
    # code block
    return value
```

- **Modules**: Python files containing functions and classes.  
- **Importing Modules**:

```python
import module_name
from module_name import function_name
```

- **Common Built-in Modules**: `math`, `random`, `datetime`, `os`, `sys`

---

## 3. Data Structures 📦

- **List**: ordered, mutable collection; supports indexing and slicing.  
- **Tuple**: ordered, immutable collection.  
- **Set**: unordered, mutable collection of unique items.  
- **Dictionary**: unordered collection of key-value pairs.  

- **Operations**:  
  - Add, remove, update elements  
  - Iterate over elements  
  - Check membership using `in` keyword

---

## 4. Algorithms (Foundations) 🔍

- Step-by-step instructions to solve problems efficiently.

- **Properties**:  
  - Correctness: solves the problem correctly  
  - Efficiency: considers time and space complexity

- **Basic Algorithm Types**:  
  - Searching: Linear Search, Binary Search  
  - Sorting: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort  

---

## 5. Dynamic Programming (DP) 🧩

- Technique to solve problems by breaking them into smaller subproblems.
- Approaches:  
  - Memoization (top-down)  
  - Tabulation (bottom-up)

- **Common Problems**:  
  - Fibonacci Sequence  
  - Coin Change  
  - Knapsack Problem  
  - Longest Common Subsequence (LCS)

---

## 6. APIs (Application Programming Interface) 🌐

- Allows interaction between different software systems.

- **Types**:  
  - REST API: Uses HTTP requests (GET, POST, PUT, DELETE)  
  - SOAP API: XML-based protocol

- **Python Libraries**:  
  - `requests`: send HTTP requests  
  - `json`: parse JSON data

- **Example Workflow**:  
  1. Send request to API endpoint  
  2. Receive response  
  3. Parse and use the data
