# Phase 2 · FastAPI + Security Basics

---

## What You'll Learn

- **Why FastAPI** — async support, automatic docs, Pydantic validation. Understand the reasons, not just the syntax.
- **Routes** — create GET and POST endpoints that return JSON.
- **Pydantic Models** — validate request bodies. Critical for AI APIs that receive unpredictable input.
- **Async Endpoints** — understand `async/await` and why it matters for AI services handling concurrent requests.
- **Parameters** — path parameters, query parameters, request bodies. Know all three cold.
- **Swagger Docs** — test your API using the auto-generated UI at `/docs`.
- **JWT Authentication** — protect endpoints with `python-jose` + `passlib`.
- **Rate Limiting** — use `slowapi` to limit requests per user. One decorator, always needed.
- **Environment Variables** — manage secrets with Pydantic Settings and `.env` files. Never hardcode API keys.
